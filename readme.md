### Nimusic

## Controls:

# Main Controls:
 - Right Stick: Move Cursor
 - A: Select Note
 - Right Bumper: Play/Pause
 - Left Bumper: Create New Note
 - Left Stick: Scroll

# When Selected Note:
 - Left Stick: Move Note
 - X & Left Stick Left/Right: Change Duration
 - X & Left Stick Up/Down: Change Frequency
 - A: Play Note
 - Start: Duplicate Note
 - B: Leave Selection
